| Annotation                     | Σκοπός                          |
| ------------------------------ | ------------------------------- |
| `@NotNull`                     | Validation στην εφαρμογή        |
| `nullable = false`             | Περιγραφή/δημιουργία του schema |
| `CHECK (credits > 0)` στη βάση | Τελική εγγύηση ακεραιότητας     |



| Τύπος πεδίου                | Validation                                           |
| --------------------------- | ---------------------------------------------------- |
| String που δίνει ο χρήστης  | `@NotBlank`                                          |
| Αριθμός που δίνει ο χρήστης | `@NotNull`, `@Positive`, `@Min` κ.λπ.                |
| Date που συμπληρώνει η βάση | Μόνο `@Column(nullable = false, insertable = false)` |
| Id                          | Τίποτα                                               |
